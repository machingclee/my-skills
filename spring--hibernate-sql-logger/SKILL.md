# Scoped Hibernate SQL Logger

This skill scaffolds a `QueryLogger` utility that scopes Hibernate SQL logging
to a single lambda block instead of enabling it globally.

## Mandatory Trigger

Invoke this skill before writing the utility when the user asks to:

- "log Hibernate SQL for one method" / "scoped SQL logging" / "QueryLogger".
- "see the SQL queries generated by this handler".
- "temporarily enable Hibernate SQL logging".
- "add scoped SQL logging to the project".

## What It Produces

```
src/main/java/{{basePackage}}/common/jpa/
  QueryLogger.java
```

## How To Use

1. Copy `templates/QueryLogger.java` into the project, replacing `{{basePackage}}`
   with the project's base package.
2. Use it in any query handler or service method:

```java
return QueryLogger.withSqlLogging(() ->
    carModelRepository.findAll().stream()
            .map(dtoMapper::toDTO)
            .toList()
);
```

## Dependencies

Requires Logback (Spring Boot's default). If the project uses Log4j2, replace
the Logback cast with `Configurator.setLevel("org.hibernate.SQL", Level.DEBUG)`.
